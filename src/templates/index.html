¿Qué Hacemos?

Antiphishing Pishing Club es un sistema basado en Machine Learning que detecta todo tipo de riesgo web, diagnosticando cada aspecto de la página y brindado un resultado detallado de cada sitio, con el fin que el usuario pueda tener una navegación segura.

Diagnóstico de Phishing Web

Nuestro sistema ayuda a detectar phishing en páginas web mediante el análisis de la estructura web.

Sistema de correo local seguro

Brindamos un servicio de correos dentro de su empresa, de esta manera los envíos y recepciones serán muy seguros.

Buen soporte y confiabilidad

Brindamos el mejor soporte en la actualidad, para resolver las dudas y problemas de nuestros clientes.

Sobre Nosotros

Somos dos estudiantes del último ciclo de la carrera de Ingeniería de Sistemas de Información de la Universidad Peruana de Ciencias Aplicadas (UPC), cuyo proyecto de tesis tiene como nombre: SISTEMA INTELIGENTE QUE UTILICE UNA TÉCNICA DE APRENDIZAJE AUTOMÁTICO (RANDOM FOREST) PARA LA DETECCIÓN DE PHISHING DE PÁGINAS WEB

Nuestros Servicios

Como equipo, brindamos servicios de detección de phishing web en dos formas, la primera mediante análisis y diagnóstico de URLs, y la otra mediante envío y recepción de correos.
